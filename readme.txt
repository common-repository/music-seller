== Description ==

	What is this?
This is a WordPress plugin that allows you to sell mp3 files / FLAC / WAV audio via PayPal in a stylish one click checkout with audio previews for each file.

	DEMO
[Full Version DEMO page](http://wp.shopfiles.com/?page_id=16797) - How the order page looks with the one click checkout and preview files.

	Full version benefits
[Full Version of Music Seller](https://www.shopfiles.com/index.php/products/music-seller-plugin-for-wordpress) - Here you can upgrade your copy to the full version, this will give you the following benefits:
	♪ Upload more than 3 files
	♪ Have audio player with previews (separate files are uploaded, for example 30 second version)
	♪ Change the color scheme of the player to match your template


	How to use this product?
It's really simple. You need to do the following steps:
1. You install the plugin.
2. Then you will in your PayPal email to receive the payments under Settings > Music Seller.
3. From there with each post or page you create you have a box where you can upload your music files (and a separate file used for preview for each of the music files for sale) and embed the cart, player and order form simply using the [music_seller] short tag.

That's it, once someone opens the page/post you added the songs, they will be able to add them to a cart and submit the order directly to PayPal and pay you. After the payment is complete the users are sent to a pre-made but customizeable thank you apge where they download their purchases. Optionally if enabled, the songs and the download links can be delivered to buyer's email.

== Installation ==

Installing abd running the plugin in 3 easy steps.

1. Upload the zip file via the plugins manager.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure PayPal account under Settings -> Music Seller

== Frequently Asked Questions ==

= Is there any limitation on how many sites I can use it? =

No tehre are no limitations.

= How does it secure my files from unauthorized downloads? =

The plugin creates a secure folder that is not accessible without making a payment for the specific file(s) first. It's impossible to steal your files for free, even the previews are well protected.
